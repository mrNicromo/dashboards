# Утилиты

Вспомогательные скрипты для команды (roadmap, импорты). Не требуются для работы дашбордов в продакшене.

| Файл | Назначение |
|------|------------|
| `roadmap_import_csv.sh` | Обёртка для загрузки CSV в Merchrules / roadmap (см. переменные в `.env.example`). |
| `cdek_weekly_sync_*.csv` | Пример выгрузки задач; можно удалить или заменить актуальной. |

Секреты для внешних API задаются через `.env` (см. корневой `.env.example`), не коммитьте токены.
